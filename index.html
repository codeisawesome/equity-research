About This Research

Each report starts with the respectable stuff: SEC filings, market statistics from sources like Yahoo Finance, and earnings call transcripts from a paid service. Then I add the less respectable but still useful stuff: structured X sentiment through Grok and YouTube sentiment through Gemini, both using custom prompts that ask for actual signal instead of the usual "vibes are bullish." That gets layered into a first-pass report, checked again against model knowledge and live web verification for discrepancies, and then turned into lightweight HTML so I can share the work with friends without emailing them a PDF the size of a misdemeanor.
